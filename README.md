# Prometheus/Grafana Workshop

This repo can be used to run the playground and workshop or just as reference.

## Target Audience

Devs and DevOps with no to intermediate experience with Prometheus/Grafana

## Requirements

* [Vagrant](https://www.vagrantup.com/)*

*(tested with Vagrant 2.2.19 using Virtualbox 6.1 on Ubuntu 20.04)

### Run

```shell
# this may take a while ...
vagrant up
```

```shell
vagrant ssh
```
